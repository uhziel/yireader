# 更新日志

遵循 语义化版本 2.0.0 (opens new window) 如何维护更新日志 1.0.0 (opens new window)

# 0.4.1

(2021-6-12)

# 变更

  • ink版，支持可以缩小到更小的阅读字体。
  • ink版章节页面里，翻页改为上1/3屏上一页，下2/3屏下一页。

# 0.4.0

(2021-5-30)

# 新增

  • 添加对Kindle等E Ink设备非标准浏览器的支持：ink版易读，访问地址http://<ip>:<端口>/ink。

# 0.3.1

(2021-3-24)

# 变更

提高易读网页重新加载时的速度

# 修正

处理书源解析章节内容有时失败的问题

# 0.3.0

(2021-3-11)

提醒

从此版本开始，依赖数据库 mongodb

# 新增

  • 允许用户添加自定义的书源、管理书源
  • 添加账号系统，方便用户跨设备间同步数据

# 0.2.1

(2020-12-29)

# 修正

  • 解决章节倒序时打开章节链接跳到不正确内容的问题

# 0.2.0

(2020-12-28)

# 新增

  • 增强对厚墨书源json格式的支持
  • 目录页面添加'正序/倒序'按钮

# 0.1.1

(2020-11-29)

# 修正

  • 修正"在 safar on ios 上点击输入框会自动放大页面"的问题

# 0.1.0

(2020-11-28)

# 新增

  • 提供阅读器的最小可行性版本。
